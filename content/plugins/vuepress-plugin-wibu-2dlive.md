## Install

```sh
npm install vuepress-plugin-wibu-2dlive
```

or

```sh
yarn add vuepress-plugin-wibu-2dlive
```

```javascript
...
module.exports = {
    ...
    plugins: [
        ...
        'vuepress-plugin-wibu-2dlive'
        ...
    ]
    ...
}
...
```
