# vuepress-plugin-baidu-autopush

> Baidu autopush SEO plugin for vuepress
> 百度站点自动推送 Vuepress 插件


## Usage

- Install

  ```sh
  npm install -D vuepress-plugin-baidu-autopush
  ```

-  Update `plugins` in `.vuepress/config.js`

  ```js
  module.exports = {
    plugins: [
      'vuepress-plugin-baidu-autopush'
    ]
  };
  ```
  or

  ```js
  module.exports = {
    plugins: {
      'vuepress-plugin-baidu-autopush':{}
    }
  };
  ```
