# vuepress-plugin-html-minify

## !!! Warning

This plugin minify output html, it may cause csr result different to ssr result.

## Usage

```bash
npm i vuepress-plugin-html-minify -D
```

Add plugin to your `.vuepress/config.js`

```js
module.exports = {
  plugins: [
    // other plugins
    ['html-minify', options],
  ],
}
```
